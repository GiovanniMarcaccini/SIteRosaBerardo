## 4. COMO SERÁ CRIADA A PÁGINA blog.html

### Estrutura base:
- Header similar ao indexvn.html (reutilizar)
- Container principal com título "Rosa Berardo na Mídia" ou "Imprensa"
- Grid responsivo:
  - Desktop: 3 colunas (`col-md-4`)
  - Tablet: 2 colunas (`col-sm-6`)
  - Mobile: 1 coluna (`col-xs-12`)
- Cada card terá:
  - Imagem (com fallback se não existir)
  - Data formatada
  - Título (link para âncora)
  - Resumo (truncado se muito longo)
- IDs de âncora: slug do título (ex: `#documentario-entre-luz-e-camera-o-audiovisual-em-leodegaria-de-jesus`)
- Footer similar ao indexvn.html

### Paths:
- Imagens: `../assets/img/imprensa/docX.jpg`
- Se imagem não existir: usar placeholder `../assets/img/fotos-para-o-site/transparent.png` ou criar fallback visual
